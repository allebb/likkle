Likkle
======

Likkle is a PHP library for the [lk2.in](http://lk2.in) URL shortening service.

Requirements
------------

* PHP >= 5.4.0

License
-------

This library is released under the [MIT license](LICENSE).

Examples
--------

A set of working examples can be found in the ``/examples`` directory.
